# Azure Terraform Projects

## Overview

This repository contains my Microsoft Azure Infrastructure as Code (IaC) projects built using Terraform as part of my cloud engineering learning journey. The projects demonstrate how to provision, configure, and manage Azure infrastructure using reusable Terraform code.

## Technologies

- Microsoft Azure
- Terraform
- Git & GitHub
- Cloud-Init
- Ubuntu Linux
- Windows Server

## Azure Resources Deployed

- Resource Groups
- Virtual Networks
- Multiple Subnets
- Network Security Groups (NSGs)
- Public IP Addresses
- Network Interfaces
- Windows Virtual Machine
- Linux Virtual Machine (Ubuntu)
- Azure Bastion
- Azure Key Vault

## Features

- Automated deployment of Azure infrastructure using Terraform
- Provisioned Windows and Linux virtual machines
- Configured Azure Bastion for secure remote management
- Used Azure Key Vault to securely retrieve VM administrator credentials
- Automated Nginx installation on the Linux VM using Cloud-Init
- Configured separate application and web subnets with dedicated NSGs
